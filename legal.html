Privacy Policy

Last Updated: November 25, 2025

1. Data Collection

DEVINE collects the following minimal data necessary for functionality:

  • **Server IDs:** To link server-specific configurations, such as custom prefixes and ignored channels.
  • **User IDs:** To identify users for premium status, playlist ownership, and administrator commands (e.g., ignoring).
  • **Voice Channel IDs:** Stored temporarily while the bot is connected to ensure continuous music service (24/7 feature).

2. Data Storage

All collected data is stored securely in an encrypted database. Data is retained for as long as the bot is active in a guild. If DEVINE is removed from a guild, all associated guild data (config, queues) is deleted within 7 days.

3. Playlist Data

Playlist data stores video URLs, not actual audio files. This data is linked to the **User ID** that created the playlist. Upon request, a user may ask for all their playlist data to be deleted.

4. Data Sharing

We do not sell or share any user or guild data with third parties, outside of necessary Discord API calls for core functionality.
